# Rinthy Website - Build Plan

## Tech Stack
- React + Vite + TypeScript + Tailwind CSS + Framer Motion + Lucide Icons

## Design Direction
- Dark premium aesthetic, Modrinth green (#1BD96A) accent
- Glassmorphism, subtle gradients
- Asymmetrical layouts, hand-tuned easing, micro-interactions
- Mobile-first, fully responsive
- Geist font
- No background gradient bubbles
